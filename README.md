
# Onfly - Booking

Listagem de hoteis disponíveis para booking, categorizados por localidade, avaliação dos usuários e preço.



## Stack utilizada

**Front-end:** Vue 3, Typescript, Quasar JS, Pinia, Vue Router, Vite.


## Instalação

Para executar o  Onfly - Booking


```bash
  yarn install
  yarn dev
```
